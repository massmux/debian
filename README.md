# Docker debian

  mantainer Massimo S. Musumeci ( massmux (at) denali.uk )

This is a docker image with ssh support. 

how to build this image?

```bash
docker build -t denali/debian .
```
then do this to run it:

```bash
docker-compose up -d
```
